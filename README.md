# Final-Assignment---Part-1-and-Part-2
 IBM Excel Basics for Data Analysis – Fleet Inventory Project

This project was completed as part of the IBM Excel Basics for Data Analysis course. The project simulates the role of a Junior Data Analyst working in a local government office, tasked with importing, cleaning, and preparing fleet inventory data for analysis.

Project Overview:
The goal of this project was to:
Import vehicle fleet inventory data from a CSV file into Excel.
Clean and standardize the dataset to improve data quality.
Prepare the data for future analysis and reporting.
This project demonstrates foundational data analytics skills using Microsoft Excel.

Business Scenario:
In this final assigment, you will be following the scenario of a recently hired Junior Data Analyst in a local government office, who has been tasked with importing some data from another department which relates to inventory information about their fleet of vehicles. The data is in comma-separated value (CSV) format and the data also needs cleaning up before you can start to run any kind of analysis on it.

Skills Demonstrated:
Importing CSV data into Excel
Data cleaning and preprocessing
Removing duplicates
Handling missing values
Standardizing text and formatting
Using Excel formulas and functions
Preparing datasets for analysis

Files Included:
fleet_inventory_raw.csv	
Original raw dataset provided for the assignment
fleet_inventory_cleaned.xlsx	
Cleaned and formatted Excel workbook

Data Cleaning Process:
The following steps were performed to clean the dataset:
Imported CSV Data
Loaded the vehicle inventory CSV file into Microsoft Excel.
Verified correct column separation and data types.
Removed Duplicates
Identified and removed duplicate vehicle records to ensure data accuracy.
Handled Missing Values
Checked for blank or incomplete fields.
Filled or standardized missing values where appropriate.
Standardized Formatting
Corrected inconsistent capitalization and spacing.
Standardized date and text formats across the dataset.
Formatted relevant fields for readability and consistency.

Key Outcomes
By completing this project, I was able to:
Transform raw CSV data into a clean, analysis-ready dataset.
Apply practical Excel data-cleaning techniques used in real-world analytics roles.
Demonstrate attention to detail and data quality management skills.
